Hopeless2D: A failure with a shorter lifespan than your IQ

This failure was made with my own last 2 brain cells. I started this when I was just 14. This garbage was the consequence of my stupid train of thought and a rubick's cube... You might wondering, "HOW THE HELL DOES A CUBE LEAD YOU TO MAKE A WHOLE FRIGGIN GAME ENGINE WITH COLLISION SYSTEM, ROTATION, PIXEL ART ENGINE, CODE EDITOR, JAVA PACKAGE, TRANSPERANCY OF ENTITIES, KEYBOARD INPUTS AND A FULLY FUNCTION RUNTIME ERROR HANDLER?!?!?!?!"...

Well the answer is quite simple... The cube made me want to render it on a screen, then I realised it was 3d and I am dumb so I randomly started scrolling through youtube because I was bored and too lazy to render a cube... Until I landed on a video which showed the concept of rendering 3D objects. Why did i suddenly feel like coding a whole rendering engine? I really don't know. A failing 3D rendering engline. I decided to make a 2D one and then convert that to a game engine. And here we are...
